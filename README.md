# css-filter-grayscale

Functional CSS for filter-grayscale

## Filesize

| File | Size |
|------|------|
| `dist/filter-grayscale.css` | 1865 bytes |
| `dist/filter-grayscale.min.css` | 1311 bytes (233 Gzipped) |

## Install

```sh
npm install css-filter-grayscale
```

## Usage

### Import

```css
@import "css-filter-grayscale";
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/css-filter-grayscale/dist/filter-grayscale.min.css">
```

### Direct

```html
<link rel="stylesheet" href="path/to/css-filter-grayscale/dist/filter-grayscale.min.css">
```

### Responsive

Responsive variants are available for each class with the following suffixes:

| Suffix | Media Query              |
|--------|--------------------------|
| `-s`   | `min-width: 32em`        |
| `-m`   | `min-width: 48em`        |
| `-l`   | `min-width: 96em`        |

Example: `.gray1-m` applies the property at the medium breakpoint and above.

## Building

```sh
npm run build
```

Processes `src/filter-grayscale.css` with [Lightning CSS](https://lightningcss.dev) and outputs to `dist/`.

- `dist/filter-grayscale.css` — formatted
- `dist/filter-grayscale.min.css` — minified
